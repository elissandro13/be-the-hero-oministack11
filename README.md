## Getting Started

git clone https://github.com/elissandro13/be-the-hero-omnistack11.git


### Prerequisites

* Node
* NPM
* React (For web)


### Installing

```
cd backend
npm install
cd ../frontend
npm install
```
### Running the Project

```
cd backend
npm start
cd ../frontend
npm start
```
## Running the tests

```
cd backend
npm test

```
